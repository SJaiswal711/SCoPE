# Gaussian Process Emulator in C

Minimal Gaussian Process emulator pipeline for cosmology PCA coefficients.

## Files

- `run_pca.c`  
  Performs PCA compression of spectra/data.

- `train_GP.c`  
  Trains exact Gaussian Process models on PCA coefficients.

- `compare.ipynb`  
  Compare C and Python GP outputs.

- `run_pca.ipynb`  
  PCA workflow notebook.

- `train_GP.ipynb`  
  GP training and testing notebook.
